# Grievance Submission Portal

This is a website where users can submit their complaints or grievances. It is built to be easy to use and reliable.

## Features

*   **Saves Your Work**: If you close the page, your progress is saved. When you come back, you can continue where you left off.
*   **Error Checking**: The form checks for mistakes as you type, so you know if you missed something.
*   **Secure**: Your data is handled safely.

## How to Run the Project

Follow these steps to set up and run the project on your computer:

1.  **Install Dependencies**:
    Open your terminal and run:
    ```bash
    npm install
    ```

2.  **Run the Server**:
    Start the project by running:
    ```bash
    npm run dev
    ```

3.  **Open in Browser**:
    Go to [http://localhost:3000](http://localhost:3000) to see the app.

## Technologies Used

*   **Next.js**: The framework for building the website.
*   **Material UI (MUI)**: For the design and look of the components.
*   **TypeScript**: To catch errors in the code.
*   **Zod**: For checking if the data you entered is correct.
